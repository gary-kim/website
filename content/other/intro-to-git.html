--- title: "Intro to Git" outputs: - reveal --- # Intro to Git by Gary Kim *** ### What is Git? ** Git is a (distributed) version control system. ** Essentially, Git keeps track of every version of your file that you have "committed". This lets you, for example, remove a change that you made that you know introduced a bug. ** Git also allows multiple people to work on the same code at the same time by allowing people to make changes on their own local systems then having code to combine changes from multiple people. This process is called merging. *** ### Pro Git 2nd Edition A lot of the information in this presentation is from the Pro Git 2nd Edition book.
